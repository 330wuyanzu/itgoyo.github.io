Mac升级到High Sierra之后VIM无法打开

Mar 23, 2018 | 阅读
在终端输入

brew uninstall vim

It requires Vim 7.3.885 or later with Lua support (“+lua”).

解决方法：
方法一：
brew install vim --with-lua
    
方法二：
brew uninstall vim
brew install luajit
brew install vim --with-luajit

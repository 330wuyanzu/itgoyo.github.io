date: ‘2017/12/15 18:03:02’ updated: ‘2017/12/18 14:38:31’
tags: - deep learning - neural network

System Preferences - Keyboard-Input Source
把所有不需要的输入法都删除掉，最后你回发现除了你留下的第三方输入法没删除，还有一个英文删除不掉。

解决方法：
点击+添加一个叫Hawaiian的输入法，然后就可以把原来删除不掉的英文输入法给删除了。大家可能会问留这个输入法和留英文输入法有什么区别吗？当然有区别，如果你保留的是系统自带的英文输入法的话，每次切换到别的页面，该页面默认输入法都会立马变成英文，即便你设置了统一源，还是这样。但是你设置了Hawaiian之后，不论你切到什么页面，该页面默认的输入法都是你现在的输入法。

最后，我在Mac上使用的是百度输入法，原因如下：

  • 输入法提示：
    • 悬浮提示
    • 窗口状态栏
  • 输入法内置了五笔

linux mint 17.3 mate桌面版
基于ubuntu14.04 LST版本

安装rime

rime的作者给出了各linux发行版下的安装方法[^1]。相比于 fcitx-rime，ibus-rime有PPA支持更新，并且安装说明比较详细[^2]，所以选择ibus-rime。

  1. 首先添加PPA：
    ubuntu中直接终端运行如下命令：
    sudo add-apt-repository ppa:lotem/rime
    linux mint 貌似无法这么添加，得从系统管理-软件源-PPA中添加ppa:lotem/rime
  2. 更新软件库
    sudo apt-get update
  3. 安装ibus-rime
    sudo apt-get install ibus
  4. 安装完成后注销重新登录系统
添加第三方词库

用了一阵子rime，发现词汇量有点少。看了一些文章，作者的意图可能不希望自带词库过大，以使输入轻快流畅，词库的扩展要靠用户输入词汇的积累。但是本人思绪很容易被打断，因此写东西时敲两三个词就碰上要翻页找对的字，太影响书写流畅性。于是果断添加词库。
百度贴吧里有很多现成词库[^3]，都是从其他输入法转换而来，可以视情况选择合适的词库，建议必要过大，够用就行，否则rime重新布置会很花时间。

  1. 首先，下载好词库，一般为.txt格式
  2. 建立一个新词典，假设新词典叫做 xxx。将下载好的词库另存到 /home/用户名/.config/ibus/rime 文件夹中，删除词组拼音阵列之外的所有内容，然后在文件最开头添加以下内容：
    ---name: xxx                    #新建词典的名字version: "0.1"sort: by_weightuse_preset_vocabulary: trueimport_tables:- luna_pinyin              #合并明月拼音词库与自定义词库---

之后保存，重命名为 xxx.dict.yaml（这是rime中词典命名格式）。

  1. 在 /home/用户名/.config/ibus/rime 文件夹中新建文本文档，编码为UTF-8，命名为 luna_pinyin.custom.yaml，这是创建一个对 luna_pinyin（明月拼音）的补丁文件，用来实现用户设置。
    在 luna_pinyin.custom.yaml 中填入如下内容：
patch:translator/dictionary: xxx     #使用新建词库 xxx 作为明月拼音的词库
  1. 点击桌面右下角状态栏中输入法图标调出菜单，再点击那个近似圆圈的图标进行出行部署。这步要花点时间，取决于你使用的第三方词库有多大。等重新部署完成，就可以开始使用输入了。
用户词库的同步

linux下也有搜狗输入法，但是貌似现在还没有用户词库的同步或导出功能，这也是 rime 的优势之一。随着输入法使用，时间越久由输入习惯产生的用户词库里词汇量就越大，把它同步备份，这样即使重装输入法，只要把用户词库同步回进程来，我们自己造的那些词就仍然在输入法词库中，不需要从新再去创造一遍。

  1. 打开文件夹 /home/bode/.config/ibus/rime 中的 installation.yaml 文件，在 installation_id： 后添加你的用户ID，在 sync_dir: 后添加用户词库的同步文件夹

  2. 终端中输入以下命令，杀死 ibus 进程：

    cd ~/.config/ibus/rime/killall ibus-daemon
3. 使用ibus-rime自带的词典管理工具 rime_dict_manager 进行用户词库同步：

/usr/bin/rime_dict_manager -s

之后会显示同步成功，在同步文件夹里就能看到备份的用户词库文件。[^1]:http://rime.im/download[^2]:https://github.com/rime/home/wiki/RimeWithIBus[^3]:http://tieba.baidu.com/p/2758001096
